## 3. 合并步骤

### 3.1 克隆 WVP 源码

```bash
cd 28181-2022
git clone https://github.com/648540858/wvp-GB28181-pro.git wvp
cd wvp
git checkout 999e6fe2  # 锁定基线版本
cd ..
```

### 3.2 执行合并

```bash
bash scripts/merge.sh
```

### 3.3 编译验证

```bash
cd wvp
mvn compile -DskipTests
```

### 3.4 指定 WVP 目录

```bash
bash scripts/merge.sh /path/to/your/wvp
```

## 4. 验证合并

```bash
bash scripts/merge.sh --verify
```

## 5. 回退合并

```bash
bash scripts/merge.sh --revert
```

## 8. 常见问题

**Q: 补丁应用失败？**
A: 确认 WVP 版本为 commit `999e6fe2`。其他版本可能行号偏移，使用 `--fuzz=3` 参数容忍少量偏移。

**Q: 编译报错找不到类？**
A: 确认所有 22 个新增文件已复制到 WVP 对应目录。运行 `bash scripts/merge.sh --verify` 检查。

**Q: 如何回退？**
A: `bash scripts/merge.sh --revert`，会反向应用补丁并删除新增文件。
